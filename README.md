Personal Portfolio Website

Overview

This is a personal portfolio website built to showcase my skills, projects, and experiences. It is designed with a modern and responsive layout, making it accessible on all devices. The website serves as a digital CV and a platform to highlight my work to potential employers or collaborators.

Features

Home Section: Introduction and professional summary.

About Me: Detailed background, skills, and education.

Projects: Showcase of projects , descriptions.

Contact Form: Users can send messages directly via the contact form.

Responsive Design: Works seamlessly on desktops, tablets, and mobiles.

Smooth Animations & UI: Clean and interactive user experience.

Technologies Used

Frontend: HTML, CSS, JavaScript
